Step 3: Design Processes to Consider

There must be a good user interface (UI) AND a good user experience (UX). UX - how user interacts with the website as a whole. Consider 1) presentation of info, 2) flow between sections, and ease of use

User experience: driven by needs and viewport of the user Methods to plan user experience: Stakeholder interview, user research, ux audits & scenario mapping, client persona

Intentional Engagement

  • Keeps user interacting with your website. Encourage purchases, or interact with ads.
  • User experience should be driven by how customer will engage with the website.
  • List what customers will be looking for when they visit the website; make it as easy as possible to get that info
